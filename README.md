# sao-datashare-plugin

> This SAO genetator will scaffold a Datashare plugin.

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao@beta
# or
npm i -g sao@beta
```

### From git

```bash
sao icij/sao-datashare-plugin MY-PLUGIN-NAME
```
